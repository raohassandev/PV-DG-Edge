# Commissioning Guide

## Before site visit

Collect:
- site name/location
- single line diagram
- device list
- communication map
- IP addresses
- RS485 wiring plan
- manuals
- Modbus unit IDs
- CT/PT ratios
- expected parameters
- user list

## Physical checks

- PC installed
- UPS connected
- Ethernet connected
- RS485 converter installed
- A/B polarity checked
- shield grounding checked
- devices powered
- comm settings known

## Software setup

1. install system
2. create super admin
3. create site
4. set timezone
5. configure backup
6. add users
7. add devices
8. import/select drivers
9. test communication
10. verify dashboard

## Device verification

For each device:
- ping or serial check
- Modbus connection test
- read known register
- compare with meter/HMI
- save evidence
- mark driver verified if values are correct
