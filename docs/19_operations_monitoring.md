# Operations and Monitoring

## System health dashboard

Show:
- API status
- acquisition worker status
- aggregation worker status
- rules worker status
- MQTT status
- PostgreSQL status
- Redis status
- CPU
- RAM
- disk
- network
- backup status

## Device communication dashboard

Show per device:
- online/offline
- last success
- last error
- success %
- timeout count
- CRC/error count
- response time
- stale metrics

## Logs

Categories:
- API logs
- worker logs
- Modbus diagnostics
- alarm logs
- audit logs
- backup logs
- system logs

## Support bundle

Engineer can export a support bundle with:
- app version
- service health
- redacted config
- device list
- driver versions
- recent errors
- system resources
- database health

Never include passwords/secrets.
